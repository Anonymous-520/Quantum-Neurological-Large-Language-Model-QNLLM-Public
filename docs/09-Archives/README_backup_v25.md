## The 4 Autonomous Task Processors

### 1. CodeLearn (Deterministic Code Analysis)

Deterministic syntax-aware task routing with O(n) complexity:

```python
from scripts.qnllm_unified_api import QNLLMFactory

processor = QNLLMFactory.create("codelearn")
processor.process_task([0.1]*50, "function_definition")
processor.process_task([0.2]*50, "loop_statement")

decision = processor.query([0.15]*50) # "function_definition"
```

**Use Cases**: Autonomous code analysis, bug detection, refactoring decisions

### 2. Strategy (Autonomous Decision Making)

Multi-level strategic autonomous processing with O(n·s) complexity:

```python
processor = QNLLMFactory.create("strategy")
processor.process_task([0.3]*50, "market_entry")
processor.process_task([0.4]*50, "pricing_strategy")

decision = processor.query([0.35]*50) # Strategic decision
```

**Use Cases**: Autonomous planning, investment decisions, strategy selection

### 3. Multimodal (Cross-Domain Autonomous Processing)

Domain detection with autonomous cross-domain reasoning O(d·n) complexity:

```python
processor = QNLLMFactory.create("multimodal")
processor.process_task([0.5]*50, "text_image_pair")
processor.process_task([0.6]*50, "audio_video_pair")

result = processor.query([0.55]*50) # Cross-domain processing
```

**Use Cases**: Autonomous healthcare analysis, autonomous finance processing

### 4. MultiAgent (Distributed Autonomous Consensus)

Distributed autonomous systems reaching consensus O(m·d) complexity:

```python
processor = QNLLMFactory.create("multiagent")

# 3 autonomous agents
agents = [QNLLMFactory.create("multiagent") for _ in range(3)]
for agent in agents:
 agent.process_task([0.7]*50, "shared_autonomy")

# Autonomous consensus emerges
decisions = [agent.query([0.75]*50) for agent in agents]
consensus = sum(d == decisions[0] for d in decisions) / len(decisions)
```

**Use Cases**: Autonomous IoT coordination, autonomous swarm systems

## Usage Examples

### Basic Autonomous Operation

```python
from scripts.qnllm_unified_api import QNLLMFactory

# Create autonomous processor
processor = QNLLMFactory.create("codelearn", input_dim=50, output_dim=5)

# Execute autonomous task
error, gate_status = processor.process_task([0.1]*50, "task_1")
print(f"Error: {error:.3f}, Gate: {'AUTONOMOUS' if gate_status else 'CONSOLIDATING'}")

# Autonomous query
decision = processor.query([0.1]*50)
print(f"Decision: {decision}")

# Verify autonomy
print(f"Task processing duty cycle: {processor.active_count / processor.step:.2%}")
print(f"Task memory size: {len(processor.task_memory)}")
print(f"Formally verified invariants: 15/15 ")
```

### Distributed Autonomous Consensus

```python
# 3 autonomous agents reaching consensus
from scripts.qnllm_distributed_consensus import run_autonomous_consensus

results = run_autonomous_consensus(
 num_agents=3,
 task_domains=["code_analysis", "strategy", "planning"],
 autonomous_steps=30
)

print(f"Autonomous agreements reached: {results['consensus_count']}")
print(f"Cross-domain consistency: {results['avg_consistency']:.3f}")
print(f"Invariant validation: {results['invariants_verified']}/15")
```

### REST API

```bash
# Start server
python deployment/qnllm_api_server.py

# Create instance (in another terminal)
curl -X POST http://localhost:8000/create -H "Content-Type: application/json" \
 -d '{"variant":"codelearn"}'

# Train
curl -X POST http://localhost:8000/train -H "Content-Type: application/json" \
 -d '{"instance_id":"ID","data":[{"input":[0.1],"label":"test"}]}'

# Query
curl -X POST http://localhost:8000/query -H "Content-Type: application/json" \
 -d '{"instance_id":"ID","input":[0.1]}'
```
